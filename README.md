## Proposal

### Create a Counter Component using Vanilla JavaScript to re-invent front end framework functionality

## Method:

- Create Shadow DOM
- Component HTML
- Render using Connected Callback Life Cycle event
- Define CustomElement and name it
- Show Web Componenton Simple Web Page
- Configure properties/attributes using ObservedAttributes callback
- Sync attributes/properties using AttrubuteChanged callback
- Refresh component on property change
- Define golden source as for property/attribute value
- Update property programmaticcally
- Handle Events within Component
